# Event Management Platform with Personalized Recommendations

Цифровая платформа для организации, управления и анализа мероприятий с системой персонализированных рекомендаций.

## О проекте

Данный проект представляет собой полнофункциональную платформу для управления мероприятиями, ориентированную как на организаторов, так и на участников событий. Система автоматизирует процессы создания мероприятий, регистрации пользователей, аналитики посещаемости, уведомлений и интеллектуального подбора событий на основе интересов пользователя.

Платформа разработана в рамках дипломного проекта и предназначена для использования в образовательной, культурной, корпоративной и event-сфере.

## Архитектура системы

Приложение построено по многослойной клиент-серверной архитектуре:

- Frontend — пользовательский интерфейс;
- Backend — бизнес-логика и REST API;
- PostgreSQL — хранение данных;
- Redis + Celery — фоновые задачи и очереди;
- SMTP сервис — email-уведомления.

## Рекомендательная система

В проекте реализован гибридный алгоритм рекомендаций, сочетающий:

- контентную фильтрацию;
- коллаборативную фильтрацию;
- анализ предпочтительных категорий;
- анализ избранных организаторов.

### Алгоритм работы

#### Контентная фильтрация
Используется TF-IDF и косинусное сходство для определения похожести мероприятий на основе:
- названия;
- описания;
- категорий.

#### Коллаборативная фильтрация
Система ищет похожих пользователей по:
- посещённым мероприятиям;
- избранным событиям;
- пользовательской активности.

#### Итоговый рейтинг
Финальный рейтинг мероприятия формируется следующим образом:

- 40% — контентная фильтрация;
- 30% — коллаборативная фильтрация;
- 20% — избранные организаторы;
- 10% — предпочтительные категории.

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/lizakat/event-management-platform.git
cd event-management-platform
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

### 3. Активация окружения

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/macOS

```bash
source venv/bin/activate
```

### 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 5. Настройка переменных окружения

Создайте `.env` файл:

```env
DATABASE_URL=postgresql://user:password@localhost/dbname
SECRET_KEY=your_secret_key

GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret

REDIS_URL=redis://localhost:6379
SMTP_EMAIL=example@yandex.ru
SMTP_PASSWORD=password
```

### 6. Запуск приложения

```bash
uvicorn app.main:app --reload
```
